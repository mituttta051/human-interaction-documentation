# Getting Started

## Preconditions

Before interacting with the system:

- define your request clearly
- provide sufficient context
- specify expected output if necessary

## Recommended Interaction Flow

1. Provide a clear request or task.
2. Add context if the task is non-trivial.
3. Optionally define the expected format of the answer.
4. Wait for response.

## Example

Input:
> Please structure this idea into a step-by-step plan.

Expected output:
> Structured, clear, and actionable plan.
